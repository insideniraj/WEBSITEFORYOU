### View Locally
1. Navigate to any website folder (e.g., `01-dental-clinic`)
2. Open `index.html` in your web browser
3. The website will display with all features working

### Deploy Online (Free Options)

#### Option 1: Netlify (Recommended)
1. Go to [netlify.com](https://netlify.com) and sign up
2. Drag and drop the website folder to deploy
3. Get a free URL like `your-site.netlify.app`

#### Option 2: GitHub Pages
1. Create a GitHub repository
2. Upload the website files
3. Enable GitHub Pages in repository settings
4. Get a free URL like `username.github.io/repo-name`

#### Option 3: Vercel
1. Go to [vercel.com](https://vercel.com) and sign up
2. Import your project folder
3. Get a free URL like `your-site.vercel.app`

## ğŸ“ Customization Guide

### Change Colors
Edit the CSS file and modify the `:root` variables:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
}
```

### Change Images
Replace image URLs in the HTML file with client's images:
```html
<img src="path/to/client-image.jpg" alt="Description">
```

### Change Content
Edit the text directly in the HTML file.
